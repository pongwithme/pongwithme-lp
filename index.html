About Pongwith.me

Pong With Me is a free multiplayer, realtime game of Pong built on theosp's implementation of Pong in which up to four players can use their smartphones as physical paddles to play. Wondering how this works? See it in action below.

We've used HTML Canvas, node.js, socket.io, express, shake.js, as well as Bootstrap for the landing page, so props to all the folks that make these technologies possible!

Think you're a l33t pong player? Mash the button below to begin playing!

Follow the ingame instructions to connect your smart phones to the game.
